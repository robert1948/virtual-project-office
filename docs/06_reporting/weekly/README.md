# Weekly Reports

Weekly operational and delivery reporting.

## Usage

- Use `docs/07_templates/WEEKLY_REPORT_TEMPLATE.md`.
- Publish one report per week with report ID and period.
- Link supporting work orders, decisions, changes, and risks.
